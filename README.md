## Learnings and Insights

### Geolocation & Leaflet.JS

Prior to this project, I did not know anything about [Leaflet.js](https://leafletjs.com/index.html) which is a JavaScript library that helps developers create interactive and mobile-friendly maps. But now, I am able to grab the current coordinates of a user through the help of Leaflet.JS & Leaflet Locate Control.

Leaflet.JS, as far as I know, just provides you with an easy interactive mobile-friendly map. The thing that helped me actually locate the user's geolocation was [Leaflet Locate Control](https://github.com/domoritz/leaflet-locatecontrol). 

The tools I got to kind of mess around with in Leaflet include:

- Markers
- Popups
- TileLayers (this helps you change the skin of the map)

In Leaflet Locate Control, I learned how to automatically locate the user's location without the need to press any buttons using the `start()` method. Then, I utilize the `stop()` method once the user's location has successfully been found.
